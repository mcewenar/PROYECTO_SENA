Proyecto final Análisis y Desarrollo de Software (ADSO) 
entrega a Guillermo Bejarano:

Calculadora de horas de trabajo.

Implementación de una aplicación web de 3 capas cuya 

1.capa de presentación (frontend) está realizada con Angular 
(lenguaje de programación Typescript y formularios reactivos). 
2. Capa intermedia del servidor (backend), 
fue realizado con Spring Boot usando Java +8 y usando el módulo JDBC 
y con endpoints para recibir peticiones desde la capa de presentación usando 
arquitectura  api-REST.
3. Capa de persistencia o servidor de base de datos, que está realizada 
en PostgreSQL sobre un contenedor en Docker para mayor facilidad.

Nota: Dentro del mismo repositorio se encuentran todas las indicaciones, reglas
de negocio y requisitos técnicos para la realización del aplicativo para la 
empresa IAS Software (caso-calculadora-handyman.pdf).
